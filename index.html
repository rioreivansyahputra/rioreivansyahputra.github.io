📱 Cara Mengganti Icon Social Media

🎯 Metode 1: Ganti dengan Foto dari Penyimpanan

  1. Klik kanan pada icon WhatsApp/Instagram/Discord → "Inspect Element"
  2. Cari kode <svg> atau <img>
  3. Upload foto Anda ke layanan seperti imgur.com atau postimg.cc
  4. Copy link foto, ganti bagian src="" dengan link foto Anda
  5. Contoh: src="https://i.imgur.com/yourphoto.png"

🔧 Metode 2: Edit Langsung di Kode

  1. Buka file HTML di text editor
  2. Cari bagian id="whatsapp-link", id="discord-link", atau id="instagram-link"
  3. Ganti seluruh <svg>...</svg> dengan:
  4. <img src="path/foto-anda.jpg" class="social-icon" alt="Icon">

📷 Tips untuk Foto Icon:

  • Ukuran ideal: 512x512px atau 1024x1024px
  • Format: PNG dengan background transparan
  • Foto bulat/persegi sesuai dengan design button
  • File size maksimal 1MB untuk loading cepat

🎮 Icon Platform yang Tersedia:

  • 🟢 WhatsApp: Chat dan komunikasi
  • 🟦 Discord: Gaming dan komunitas
  • 🌈 Instagram: Foto dan social media

🚀 Shortcut Cepat:

Tekan Ctrl + Shift + I untuk membuka Developer Tools, lalu edit langsung di browser untuk testing!
